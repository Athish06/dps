Keyed Columnar Cipher Parameters

Column Read Order

Enter the order in which columns should be read (1-indexed). This is auto-calculated from the keyword, but you can customize it.

Keyed Columnar Transposition:
  • Keyword: Determines the number of columns and read order
  • Column Order: Alphabetical ranking of keyword letters (ties: left-to-right)
  • Encryption: Write plaintext row-by-row, read columns in order
  • Decryption: Fill columns with ciphertext, read row-by-row

Example: ZEBRAS ‚Üí Order: 6,3,2,4,1,5 (A=1, B=2, E=3, R=4, S=5, Z=6)
