Rail Fence Cipher Parameters

Transposition cipher that writes text in a zig-zag pattern across multiple rails, then reads row by row.

Rail Fence Cipher:
  • Encryption: Write plaintext diagonally down and up in a zig-zag pattern, then read each rail left to right
  • Decryption: Calculate characters per rail, fill rails with ciphertext, read in zig-zag order
  • Key: Number of rails determines the pattern depth
